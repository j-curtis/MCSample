MCSample

A class for generating multidimensional samples from a specified PDF.

Conventions

Functions are named using camel text with the first letter of each word lowercase while subsequent letters of each new word are capitalized. As an example, myFunctionName().

Variable Names

A variable is identified first by a key identifying the type, then _unique_variable_name with words lowercase, separated by underscores. The type keys are:

  • d_ ==> double
  • i_ ==> integer
  • b_ ==> boolean
  • pd_ ==> pointer to/array of doubles
  • pF_ ==> pointer to function

Class Attributes

Protected:

int i_dimension -- An integer indicating the dimension of the sample space and the vectors sampled.
double * pd_lower_lim -- An array of size i_dimension that stores a list of lower bounds on the sample space.
double * pd_upper_lim -- An array of size i_dimension that stores a list of upper bounds on the sample space.
double (*pF_pdf)(double *) -- A pointer to a function of return type double and argument type double * which is the PDF we are drawing from.

Public:

static double getRandomInRange(double d_lower, double d_upper) -- A static method for generating a random double in [d_lower,d_upper).
MCSample(inti_dim,double *pd_lower,double *pd_upper,double (*pF_func)(double *)) -- Constructor. Requires dimensionality of sample space, then list of lower and upper bounds, then a pointer to a PDF to draw from.
double *genSample() -- Returns a vector randomly drawn from the probability PDF passed to the generator.

Usage Notes

This implementation is extremely crude; it heavily relies on unsafe usage of arrays and pointers. Future updates will hopefully employ more robust objects. In addition, future updates will hopefully replace usage of a function pointer with a function object.
